Reviews Sentiment Analysis with Recurrent Neural Networks

NLP • SimpleRNN • Model Evaluation

Built an NLP sentiment classifier that cleans and tokenizes review text into sequences, then trains a SimpleRNN model to predict sentiment.

The final model achieved 84% test accuracy, but a key limitation was the dataset labelling: sentiment was inferred from numeric ratings rather than manually annotated labels, introducing noise (e.g., positive text with non-maximum ratings).

Originally, ratings ≥ 3.5 were labelled positive (72% test accuracy). Refining labels to positive ≥ 4 and negative ≤ 3 improved test accuracy to 84%, showing earlier underperformance was largely due to inconsistent labels rather than model architecture.

Neural Network Implementation from Scratch (Python & NumPy)

MLP • Backpropagation • Fashion-MNIST

Designed and implemented a fully connected three-layer Neural Network (MLP) entirely from scratch (no deep learning frameworks). Implemented forward propagation, backpropagation, weight updates, and loss calculation using only Python and NumPy.

Trained on the Fashion-MNIST dataset (70,000 grayscale images across 10 categories) and achieved a peak 87.65% test accuracy. Optimised performance using NumPy vectorisation (np.dot) and built a full training pipeline with Mini-batch SGD and Sigmoid activation for stable convergence.
